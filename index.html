Semana 9 · Material de Clase (Consultas simples y complejas en SQL)

Descarga directa de los archivos de práctica. Puedes abrir esta página en clase y pedir a los estudiantes que hagan clic en el archivo correspondiente a su herramienta.

MySQL / XAMPP (phpMyAdmin)

Importa este script en la base semana9_demo desde la pestaña Importar de phpMyAdmin.

Descargar: semana9_demo_mysql.sql 
SELECT VERSION() AS mysql_version;
SELECT COUNT(*) AS clientes FROM clientes;

SQLite / DBeaver

Abre el archivo como conexión SQLite en DBeaver.

Descargar: semana9_demo.sqlite 
SELECT sqlite_version() AS sqlite_version;
SELECT COUNT(*) AS clientes FROM clientes;
